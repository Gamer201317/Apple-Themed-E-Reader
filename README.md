Elpa Book

Elpa Book is a low-cost, offline e-reader designed for reading personal books without subscriptions, accounts, or cloud dependency.

It focuses on simplicity, control, and fully offline usage.



Why this project?

Most modern e-readers are tied to ecosystems, accounts, or subscription services. Elpa Book is designed as an alternative that gives full ownership back to the user: your files, your device, no external dependencies.



Features

- Fully offline e-reader (no internet required)
- Supports local files via SD card
- Low-cost hardware design
- Customizable enclosure (3D printed or DIY)
- Minimal and simple electronic design
- Touchscreen-based navigation


Hardware

- **Microcontroller:** Raspberry Pi Zero 2 W  
- **Display:** 7-inch 1024×600 capacitive touchscreen  
- **Storage:** MicroSD card (for OS + books + files)  
- **Input:** Touch interface  
- **System:** Lightweight custom Linux-based setup



How it works

Elpa Book runs on a Raspberry Pi Zero 2 W connected to a 7-inch touchscreen display. Books are stored locally on a microSD card and accessed directly through a simple offline interface.

The system is designed to automatically boot into a reading environment, allowing users to select and open books without requiring any internet connection.



Readify

Readify is a companion web tool built for managing books and files for the Elpa Book. It is fully free to use and designed to simplify transferring and organizing content for the device.



 Customization

Elpa Book is designed to be modular and customizable:

- Expandable storage via SD card replacement
- Adjustable battery capacity depending on build
- Fully custom 3D printed enclosure design
- Optional hardware modifications depending on user needs
